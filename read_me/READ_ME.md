# Mahdere Selam Community Portal 🇪🇹 ⛪

A premium, full-stack web application designed for the Mahdere Selam Ethiopian Orthodox Tewahedo Church. This platform modernizes parish administration and provides a centralized hub for youth education and community resources.

---

## The Vision

The goal of this project is to provide our congregation with a digital experience that reflects the depth of our tradition. By moving beyond static pages, this portal empowers youth leaders and provides students with a dedicated space for theological discussion.

## Tech Stack

- **Frontend:** React + Vite
- **Styling:** Tailwind CSS + shadcn/ui
- **Backend:** Node.js + Express
- **Database:** MongoDB
- **Authentication:** JWT (JSON Web Tokens)

## Church Schedule & Features

The application is designed around our specific liturgical life:

- **Liturgy Hub:** Clear, dynamic scheduling for 5 AM - 11 AM services.
- **Youth Ministry Portal:** Dedicated sections for Kids (Bible stories) and Teens (Hymns/Moral guidance).
- **College Bible Study:** A resource archive for slideshows and theological discussions.
- **Admin Dashboard:** A secure backend for church leaders to update weekly announcements.
